# Selenium Language Tests

Этот репозиторий содержит автотесты для проверки интерфейса интернет-магазина на разных языках.

## Описание проекта

- Тесты написаны на Python с использованием Selenium WebDriver
- Проверяют корректное отображение элементов интерфейса на разных языках
- Поддерживаются: английский, испанский, французский, русский и другие языки

## Запуск тестов

```bash
# Установка зависимостей
pip install selenium pytest

# Запуск тестов
pytest --language=es test_items.py    # Испанский
pytest --language=fr test_items.py    # Французский  
pytest --language=ru test_items.py    # Русский
